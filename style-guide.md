# 永盛工程行 UI 設計規範 (Style Guide)

本文件定義了永盛工程行網站的統一視覺風格與組件規範，確保所有頁面在 PC 與移動端具備一致的專業體驗。

## 1. 核心配色方案 (Color Palette)

| 類型 | 顏色代碼 | 用途 |
| :--- | :--- | :--- |
| **主色 (Primary)** | `#0066b3` | 按鈕、圖示、強調文字、導航懸停 |
| **深色 (Deep Blue)** | `#002b5c` | 按鈕懸停、標題、強烈強調 |
| **強調色 (Accent)** | `#FFCC1A` | 提示框邊框、重要按鈕、重要圖示 |
| **文字主色** | `#1e2b3c` | 內文、副標題 |
| **文字副色** | `#5a6e85` | 說明文字、圖片說明 |
| **背景色** | `#ffffff` | 頁面主要背景 |
| **交替背景** | `#f7f9fc` | 區塊間隔、背景區域 |

## 2. 字體與排版 (Typography)

- **字體族**: `'Inter', sans-serif`
- **行高**: `1.6` (內文), `1.2` (標題)
- **標題規範**:
  - `H1 / Section Title`: `3.2rem` (PC), `2.4rem` (Mobile), 粗體 (700)
  - `H2 / Sub Title`: `2.2rem` (PC), `1.8rem` (Mobile), 粗體 (600)
  - `Section Subtitle`: `1.25rem`, 文字副色

## 3. 組件設計規範

### 按鈕 (Buttons)
```html
<a href="#" class="btn btn-primary">立即預約</a>
```
- **樣式**: 圓角 `50px`, 粗體, 陰影 `0 10px 20px rgba(0,102,179,0.15)`
- **交互**: 懸停時向上移動 `3px`, 陰影加深, 顏色變為深藍色。

### 卡片 (Cards)
```html
<div class="info-card">
  <i class="fas fa-tools"></i>
  <h3>專業技術</h3>
  <p>描述文字內容...</p>
</div>
```
- **樣式**: 圓角 `32px`, 背景白色, 邊框 `1px solid #edf2f7`
- **交互**: 懸停時向上移動 `10px`, 產生柔和陰影 `0 30px 60px rgba(0,0,0,0.06)`, 邊框變藍。

### 圖片 (Images)
- **樣式**: 圓角 `32px`, 陰影 `0 20px 40px rgba(0,0,0,0.08)`
- **說明**: `img-caption` 文字居中, 斜體, 副色。

## 4. 柵格系統與間距 (Grid & Spacing)

- **容器**: `max-width: 1400px`, 左右內距 `40px` (PC) / `24px` (Mobile)
- **區塊間距**: `padding: 100px 0` (PC) / `80px 0` (Mobile)
- **網格**: 
  - `grid-2`: 2 欄佈局, 間距 `80px`
  - `grid-3`: 3 欄佈局, 間距 `40px`

## 5. 導航與頁腳

- **導航列**: 白色背景, 底部細邊框, 固定頂部。
- **移動選單**: 點擊漢堡圖示切換 `active` 類別, 下拉選單點擊展開。
- **頁腳**: 深色背景 `#0f1a2a`, 四欄佈局, 底部版權區塊。

## 6. 交互腳本規範

所有頁面必須包含統一的 `fetch` 載入邏輯，並在載入後初始化移動端事件綁定。
